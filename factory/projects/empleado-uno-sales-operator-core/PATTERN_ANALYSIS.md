# PATTERN_ANALYSIS — Sales Operator Core

Patterns to reuse:
- Agent Core functional module pattern.
- CRM Core invisible CRM pattern.
- Sales Funnel Core stage/cadence/stop-rule model.
- Voice/Vapi adapter pattern for calls/SMS where configured.
- Factory increment + evidence gates.

Anti-patterns:
- Astroturfing/fake organic traffic.
- Spam/outreach without policy gates.
- JSON-only operational state.
- Separate CRM source of truth.
